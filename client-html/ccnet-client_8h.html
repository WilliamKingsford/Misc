enum CcnetClientMode

CcnetClientMode: : Synchronous mode : Asynchronous mode

CcnetClient can run in synchronous or asynchronous mode. In synchronous mode, every function call to CcnetClient is blocked until the ccnet daemon returns. In asynchronous mode, function calls are not blocked, and the user should use the processor mechanism to interact with the daemon.
