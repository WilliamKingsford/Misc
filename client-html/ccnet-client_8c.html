SECTION:ccnet-client : The basic class to interact with the ccnet daemon : CcnetClient ccnet.h

CcnetClient is the basic class to interact with the ccnet daemon. CcnetClient can work in two different mode, i.e., CCNET_CLIENT_SYNC and CCNET_CLIENT_ASYNC (See CcnetClientMode). The two modes have different sets of APIs.
